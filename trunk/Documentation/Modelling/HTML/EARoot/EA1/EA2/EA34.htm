-- Definition -- <br /></p><p>The area definition feature refers to the geometrical data of the area of a defined unit. <br /></p><p><br /></p><p>Using ISO 19125 (Simple Feature Geometry), an area representation is described by a 'MultiSurface' object, which represents a collection of one or more polygons. each polygon is described by a collection of 'Curves', e.g., a 'LinearRing' describing a polygon.. <br /></p><p><br /></p><p>-- ISO 19125 vs ISO 19107--<br /></p><p>Individual SF geometry types correspond to one or more goemetry types of spatial schema, e.g., SF geometry collection (e.g., MultiSurface) corresponds to the GM_Aggregate of the spatial schema.<br /></p><p><br /></p>
