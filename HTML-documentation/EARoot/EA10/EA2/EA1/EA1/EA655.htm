Appears In: Vertical Obstruction
-- Description --<br /></p><p>All fixed (whether temporary or permanent) and mobile objects, or parts thereof that extend above the surface of the Earth. Those vertical structures, referred to as obstacles, that extend above a defined surface intended to protect aircraft in flight.<br /></p><p><br /></p><p>-- Definition --<br /></p><p>A vertical obstruction is defined as a unit, created and guarded by a named authority, e.g., Optus for mobile towers, they have 'versions' for being created, moved, removed etc. Each vertical obstructions comes with a 'geometry representation' to describe the horizontal extent as either being a point, a curve or a polygon. All definitions and behaviors are inherited from the transport common model, which imports the core definition model.<br /></p>
