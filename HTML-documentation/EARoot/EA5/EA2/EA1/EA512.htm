ISO 19112 "Spatial Referencing by Identifier" standardizes a model for Gazetteer information, based on the assumption that a LocationInstance has:<br /></p><p>- one preferred Identifier  <br /></p><p>- a competent adminstrator<br /></p><p>- a location-type (i.e. feature type) that classifies the location instance<br /></p><p>- a spatial extent, or a representative point location, or both<br /></p><p>- optionally, a temporal extent (period of validity)<br /></p><p>- optionally, multiple alternative identifiers or names by which it is known<br /></p><p>and is included in one or more Gazetteers. <br /></p><p><br /></p><p>This basic set of requirements serves as a checklist that any conformant Gazetteer implementation must satisfy. <br /></p><p><br /></p><p>The FSDF Place Names model should support all requirements from ISO 19112. <br /></p><p><br /></p><p>The mapping-model should indicate how each of the named classes, attributes and association roles from ISO 19112 are realised in FSDF Gazetteer. <br /></p>
