A GML implementation of ISO 19112 is used in the Gazetteer Profile of WFS ("WFS-G") to provide a standard Gazetteer service interface to which a WFS client can connect. The GML implementation closely follows the ISO 19112 conceptual schema, with minor changes due to XML Schema limitations. A UML model can be reverse engineered from the GML schema, providing a convenient representation of the standard WFS-G gazetteer payload. <br /></p><p><br /></p><p>The CGNA has deployed a WFS-G Gazetteer service for Australia, managed and hosted by GA. The payload of the WFS-G deployed by CGNA should be understood as an <i>implementation </i>of the FSDF Place Names model. <br /></p><p><br /></p><p>The mapping model should indicate how the mandatory and optional classes, attributes and association roles from FSDF Gazetteer are implemented in WFS-G. <br /></p>
