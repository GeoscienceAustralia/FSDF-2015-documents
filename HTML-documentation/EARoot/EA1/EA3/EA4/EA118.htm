-- Name --<br /></p><p>baseline<br /></p><p><br /></p><p>-- Definition --<br /></p><p>The line from which the outer limits of the territorial sea and certain other outer limits are measured.<br /></p><p><br /></p><p>-- Description --<br /></p><p>The baseline comprises of one or more baseline segments - the baseline segment can be normal, straight or archipelagic depending on the method used to determine the baseline segment. <br /></p><p><br /></p><p>COMMENT: For this moment there is no clearly identified use-case that would require Baseline spatial object types, therefore it is not strictly necessary to include these objects in the dataset. <br /></p>
