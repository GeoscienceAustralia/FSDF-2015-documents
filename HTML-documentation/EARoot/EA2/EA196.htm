The FSDF Core Definition Model package contains common behavior model for any unit definitions, including nested hierarchy of areas.  The model addresses issues of navigation and cross referencing between hierarchies and the scoping of topology to versions of hierarchies.<br /></p><p><br /></p><p>The hierarchy describes the topological relationships, not the (potentially multiple) alternative geometric descriptions of objects (see package AreaRepresentation).<br /></p>
