-- Definition --<br /></p><p>Version definition refers to a gazetted geometry defined by a nominated process; it handles changes in the boundary definition with reference to temporal data of the creation, change, deletion or redistribution of a defined area. <br /></p><p><br /></p><p>An version definition can belong to one or more defined units and only refers to a maximum of one geometrical representation. <br /></p>
